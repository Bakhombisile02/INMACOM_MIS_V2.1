# Changelog

All notable changes to the INMACOM MIS application are tracked here.

---

## V2.0.0 - Iteration 2 (Current)
### Added
- **IIMA Compliance Dashboard:** Monitors instream flow (Eflow) requirements and water allocation thresholds across Tripartite basins.
- **GIS Review Flow (2-Step Verification):** Submitters upload water flow/quality measurements; approvers (admins/managers) approve or reject with comments.
- **Polymorphic Commenting:** Shared comment threads on stations, revisions, measurements, and disasters with `@mentions` notification system.
- **Document Library:** Shared file server supporting folder hierarchy, policy uploads, and tri-nation document sharing.
- **Audit Logs:** Database audit logger tracking all station configurations and measurement reviews.

---

## V1.0.0 - Iteration 1
### Added
- **Base Stations Registry:** Core database tables for stations tracking in Mozambique, South Africa, and Eswatini.
- **Firebase Auth Bridge:** Basic signup/login flows utilizing the Firebase Client SDK.
- **Basic GIS Mapping:** Map view showing station coordinate coordinates.
