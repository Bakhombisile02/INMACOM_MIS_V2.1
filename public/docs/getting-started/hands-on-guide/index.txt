8:T6cb,# Hands-on User Guide

This guide is designed for INMACOM MIS end-users—including **Data Entry Clerks**, **Water Resource Managers**, and **System Administrators**—to walk through the actual day-to-day operation of the platform.

Here, you will find step-by-step instructions with real application screenshots demonstrating how to use each feature of the system.

---

## Roles & Responsibilities

The system is configured with three distinct user roles:

| Role | Primary Responsibilities |
| :--- | :--- |
| **Data Entry Clerk** | Submits new live measurements (flow levels, dam levels, water quality, etc.) and suggests station revisions. |
| **Water Resource Manager** | Reviews and approves/rejects pending measurements, configures compliance thresholds, and updates allocation rules. |
| **System Administrator** | Manages user registrations, resets passwords, configures global settings, and conducts audit exports. |

---

## Core Operational Workflows

Navigate through the user guide using the sidebar:
1. [**Access & Navigation**](/getting-started/hands-on-guide/access-and-navigation): Logging into the system and reading the geographic dashboard.
2. [**Station Management**](/getting-started/hands-on-guide/station-management): Creating and revising transboundary water stations.
3. [**Data Entry & Review**](/getting-started/hands-on-guide/data-entry-and-review): The 2-step verification review flow for daily measurements.
4. [**Compliance & Disasters**](/getting-started/hands-on-guide/compliance-and-disasters): Setting instream flow requirements and logging hazard notifications.
